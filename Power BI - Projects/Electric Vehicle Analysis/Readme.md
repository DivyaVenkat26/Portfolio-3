# Electric Vehicle (EV) Analysis Dashboard
## Project Overview

This project features an interactive Power BI dashboard analyzing EV data to uncover insights on adoption trends, top manufacturers, model performance, and CAFV eligibility.
The goal is to support data-driven decisions in the electric vehicle sector.

## Data Preparation

Imported dataset using Power Query and cleaned/verified data.

Removed non-informative columns (VIN (1-10), County).

Adjusted data types and ensured no missing or invalid values.

## Key Metrics (DAX Measures)

Total Vehicles: 150k

Average Range: 67.83 km

Total BEV Vehicles: 117k (78%)

Total PHEV Vehicles: 34k (22%)

## Dashboard Highlights

Visuals: Area Chart (Vehicles by Year), Map (by State), Bar Chart (by Make), Donut Chart (CAFV Eligibility), Treemap (by Model)

Slicers: City, EV Type, State for dynamic exploration

Layout: Interactive pages to explore trends, manufacturers, models, and eligibility

## Insights

Top Manufacturer: Tesla (68,943 vehicles)

Best-selling Models: Tesla Model Y, Model 3, Nissan Leaf

Sales Trends: Steady growth since 2020; peak sales in 2023 (37,079 vehicles)

Top State: Washington (150,082 vehicles)

CAFV Eligibility: 41.81% eligible, 11.6% not eligible, 46.33% unknown

## Tools Used

Power BI | Power Query | DAX
